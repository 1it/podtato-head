## Deployment

### Create Project
```
./initProject create-project
````

### Onboard Service
```
./initProject onboard-service
````

### Deploy Service (new-artifact)
```
./initProject deploy-service
````

### Upgrade Service (new-artifact)
```
./initProject upgrade-service
````
